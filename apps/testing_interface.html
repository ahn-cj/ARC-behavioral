Task Instructions

Before starting the study, please make sure your environment is distraction-free, and put your browser is in fullscreen mode.

On each task, you will see some grids with colored pixels to the left of the screen (Task Demonstration Grids), a large grid with colored pixels in the middle of the screen (Main Input Grid), and an empty grid to the right of the screen (Main Output Grid).

The Task Demonstration Grids are example pairs of input-output solutions. Your job is to figure out the relationship between input and output from the Task Demonstration Grids, and use that knowledge to fill out the Main Output Grid based on the pattern in the Main Input Grid.

To fill out the Main Output Grid, select a pixel color from the color picker below the grid, and click on pixels to color them in. You have the option to copy over the pixels from the Main Input Grid to the Main Output grid by clicking on the "Copy from Input" button. Clicking on the "Reset Grid" button will unfill all pixels in the Main Output Grid. Once you have completed your solution, click on the green "Submit" button.

If your solution is correct, you will proceed to the next task. If it is incorrect, you have up to 3 tries to get the correct solution. Solve the tasks as quickly but accurately as possible.

Click "Start" to begin.
